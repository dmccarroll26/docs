A condensed reference for codes, hardware actions, and support contacts. Use this when you know what you need and just want the quick answer.

## Five.ID terminal

| Action | How to |
| --- | --- |
| Reset card reader | Insert a pin into the reset hole on the side of the reader |
| Access Wi-Fi settings | Tap home screen 3 times quickly → Password: `GiveMeFive!` → Open WiFi Settings |
| Exit settings | Long-press the FLIC button, or unplug and replug the terminal |
| Correct power socket | Always use the socket marked with a Five sticker |
| QR code scanning distance | Hold QR code 10–20 cm from the sensor |
| Five.ID Support | support@five.id · +44 7916 468 891 |

## Kurve / Elo kiosk

| Action | How to |
| --- | --- |
| Soft restart | Hold the Elo power button (rear/underside) for 3 seconds → wait 10 seconds → press once |
| Hard restart | Unplug power → wait 30 seconds → replug |
| Access Kurve admin panel | Hold the screensaver for 3 seconds → passcode `925790` |
| Lock kiosk (out of order) | Admin panel → lock the kiosk |
| Access card machine settings | Supervisor code `2013` |
| Check card machine Wi-Fi | Supervisor code `2013` → Wi-Fi / Network settings |
| Not fixable in 5 minutes? | Lock via admin panel → Out of Order sign → direct to till → escalate |
| Kurve Support | WhatsApp: +44 117 325 3256 |
