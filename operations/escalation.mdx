If you've followed the platform troubleshooting steps and the issue isn't resolved, escalate using the contacts below.

<Note>
  **Have ready before escalating:** store name, time the issue started, what you've already tried, and any error messages (photos help).
</Note>

## Contacts

| Contact | Details | Responsibility | When to call |
| --- | --- | --- | --- |
| Dave (Ops / Tech) | First point of contact | All POS, kiosk, network, terminal, and tech issues | Any issue not resolved by this guide |
| Area Manager | Lee / Renan | Store closure decisions, staffing impact, customer comms | Store needs to close or trade severely impacted |
| Five.ID Support | support@five.id · +44 7916 468 891 | Payment terminal hardware faults, transaction disputes, replacement | Directed by Dave, or if Dave unavailable and terminal is down |
| Kurve Support | WhatsApp: +44 117 325 3256 · Or via Kurve dashboard | Kiosk software, menu sync, order routing, server outages | Directed by Dave |
| Lightspeed Support | Via Lightspeed dashboard | POS software bugs, server outages, account issues | Only if directed by Dave |
| Internet Provider | See store setup sheet | Broadband outages, router replacement | Only if directed by Dave |

## Need help?

If trade is severely impacted or the store may need to close, see the [Store closure checklist](/operations/store-closure).
