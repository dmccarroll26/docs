Issues with the QR code scanner used for Leat loyalty.

### QR code scanning not working (Leat loyalty)

<Steps>
  <Step title="Position the code">
    Hold the QR code 10–20 cm from the sensor.
  </Step>
  <Step title="Brighten the screen">
    Customer's phone screen brightness up, code clearly displayed.
  </Step>
  <Step title="Clean the lens">
    Dry cloth.
  </Step>
  <Step title="Manual fallback">
    Enter the customer's loyalty details in Leat using their phone number.
  </Step>
</Steps>
