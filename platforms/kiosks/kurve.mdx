Sprout kiosks run **Kurve** software on Elo I-Series 22" Android touchscreens. This page covers software issues. For hardware problems (no power, screen flickering, touch calibration), see [Elo I-Series](/platforms/kiosks/elo-i-series).

<Tip>
  **For all kiosk errors:** Take a photo or screenshot of the error and send it to Kurve Support via WhatsApp (`+44 117 325 3256`) with the store name and time. Do this **first**, before troubleshooting, so Kurve can begin investigating.
</Tip>

<Note>
  The kiosk card machines have their own Wi-Fi connection, separate from the kiosk screen. **Both** must be on the store Wi-Fi for payments to work. Card machine settings require supervisor code `2013`.
</Note>

## Kiosk screen frozen or unresponsive

<AccordionGroup>
  <Accordion title="Kurve app frozen / screen not responding to touch">
    <Steps>
      <Step title="Wait briefly">
        15–20 seconds — may be processing or syncing.
      </Step>
      <Step title="Open the admin panel">
        Hold screensaver/home for 3 seconds → passcode `925790`.
      </Step>
      <Step title="Soft restart">
        Hold the Elo power button (rear/underside) for 3 seconds.
      </Step>
      <Step title="Power back on">
        Wait 10 seconds, press power once.
      </Step>
      <Step title="Relaunch Kurve">
        Tap the Kurve icon if it doesn't auto-launch.
      </Step>
      <Step title="Hard restart">
        Unplug power, wait 30 seconds, replug.
      </Step>
    </Steps>
  </Accordion>
  <Accordion title="Kiosk displaying an error message">
    <Steps>
      <Step title="Take a photo">
        Capture the error.
      </Step>
      <Step title="Send to Kurve Support">
        WhatsApp `+44 117 325 3256` with store name and time.
      </Step>
      <Step title="Note the time">
        Record exactly when it appeared.
      </Step>
      <Step title="Restart Kurve">
        Admin panel via passcode `925790`.
      </Step>
      <Step title="Lock the kiosk">
        Prevent customer use while troubleshooting.
      </Step>
      <Step title="Hard restart Elo">
        Hold power 3 seconds, wait, press once.
      </Step>
      <Step title="Escalate with photo">
        Lock kiosk, post Out of Order sign, contact Dave.
      </Step>
    </Steps>
  </Accordion>
</AccordionGroup>

## Payments not processing

<AccordionGroup>
  <Accordion title="Card machine showing “No Connection”">
    <Steps>
      <Step title="Check store Wi-Fi">
        See [Network & Wi-Fi](/operations/network).
      </Step>
      <Step title="Check the card machine separately">
        It connects independently of the kiosk screen.
      </Step>
      <Step title="Enter supervisor code">
        Code: `2013`.
      </Step>
      <Step title="Verify network">
        Check Wi-Fi / Network settings.
      </Step>
      <Step title="Reconnect">
        Choose the correct store network.
      </Step>
      <Step title="Test a payment">
        Exit and run a test.
      </Step>
      <Step title="Restart if needed">
        Unplug, wait 15 seconds, replug.
      </Step>
    </Steps>
  </Accordion>
  <Accordion title="Payment times out or fails on kiosk">
    <Steps>
      <Step title="Retry">
        Customer tries again.
      </Step>
      <Step title="Check Wi-Fi">
        Supervisor code `2013` to inspect.
      </Step>
      <Step title="Restart the router">
        If multiple payments fail. See [Network & Wi-Fi](/operations/network).
      </Step>
      <Step title="Redirect to till">
        If failures continue.
      </Step>
      <Step title="Escalate">
        Contact Dave after 15 minutes.
      </Step>
    </Steps>
  </Accordion>
  <Accordion title="Contactless not working on kiosk card machine">
    <Steps>
      <Step title="Clean the area">
        Dry cloth.
      </Step>
      <Step title="Hold steady">
        Card flat for 2–3 seconds.
      </Step>
      <Step title="Restart the machine">
        Unplug, wait 15 seconds, replug.
      </Step>
      <Step title="Use chip & PIN or main till">
        If contactless still fails.
      </Step>
      <Step title="Log for escalation">
        Track the issue.
      </Step>
    </Steps>
  </Accordion>
</AccordionGroup>

## Orders not reaching the kitchen

<AccordionGroup>
  <Accordion title="Customer completes order on kiosk but kitchen doesn't receive it">
    <Steps>
      <Step title="Check the Active iPad first">
        See [Lightspeed → Active iPad](/platforms/lightspeed#the-active-ipad-local-server). It bridges kiosk and docket printer.
      </Step>
      <Step title="Are orders on the Active iPad?">
        If yes but not printing, see [Instore Printers](/platforms/instore-printers#digital-line-docket-printer).
      </Step>
      <Step title="Are orders missing entirely?">
        Kiosk-to-POS issue. Check kiosk Wi-Fi.
      </Step>
      <Step title="Same network">
        Kiosk and Active iPad must share the store Wi-Fi.
      </Step>
      <Step title="Check Kurve back office">
        For queued or failed orders.
      </Step>
      <Step title="Restart the Elo">
        Test with a new order.
      </Step>
      <Step title="Workaround">
        Direct customers to the main till; post Out of Order sign.
      </Step>
      <Step title="Escalate">
        Dave / Kurve support.
      </Step>
    </Steps>
  </Accordion>
  <Accordion title="Orders arriving with missing items or wrong details">
    <Steps>
      <Step title="Don't fix at store level">
        Kurve menu-sync issue.
      </Step>
      <Step title="Take a photo">
        Capture KDS/docket and kiosk screen.
      </Step>
      <Step title="Send to Kurve Support">
        WhatsApp `+44 117 325 3256` and escalate to Dave.
      </Step>
      <Step title="Take orders at till">
        Until resolved.
      </Step>
    </Steps>
  </Accordion>
</AccordionGroup>

## Wi-Fi / network issues

<AccordionGroup>
  <Accordion title="Kiosk shows offline or can't connect">
    <Steps>
      <Step title="Check other devices">
        Confirm whether store Wi-Fi is up.
      </Step>
      <Step title="Open admin panel">
        Hold home for 3 seconds → passcode `925790`.
      </Step>
      <Step title="Reconnect Wi-Fi">
        Choose the correct store network.
      </Step>
      <Step title="Lock the kiosk">
        Prevent customer use while troubleshooting.
      </Step>
      <Step title="Restart Kurve">
        After reconnecting.
      </Step>
      <Step title="Check the card machine separately">
        Supervisor code `2013`.
      </Step>
    </Steps>
  </Accordion>
  <Accordion title="Connected to Wi-Fi but Kurve won't load / shows errors">
    <Steps>
      <Step title="Check internet">
        Test on another device.
      </Step>
      <Step title="Possible Kurve outage">
        If internet works, escalate.
      </Step>
      <Step title="Restart Elo">
        Hold power 3 seconds, wait, press once.
      </Step>
      <Step title="All kiosks affected?">
        Likely server-side. Direct customers to till and escalate.
      </Step>
    </Steps>
  </Accordion>
</AccordionGroup>

<Tip>
  **Kiosk golden rule:** If a kiosk isn't working and you can't resolve it within 5 minutes, lock it via the admin panel (hold screensaver 3 seconds → passcode `925790`), post an Out of Order sign, and direct customers to the main till. Don't spend service time troubleshooting — escalate and serve customers.
</Tip>

## Quick reference

| Action | How to |
| --- | --- |
| Access Kurve admin panel | Hold the screensaver for 3 seconds → passcode `925790` |
| Lock kiosk (out of order) | Admin panel → lock the kiosk |
| Access card machine settings | Supervisor code `2013` |
| Check card machine Wi-Fi | Supervisor code `2013` → Wi-Fi / Network settings |
| Not fixable in 5 minutes? | Lock via admin panel → Out of Order sign → direct to till → escalate |
| Kurve Support | WhatsApp: +44 117 325 3256 |
